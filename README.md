## 3. 데이터

### - 수집

**출처**: [국가법령정보센터](https://www.law.go.kr)

`Playwright` 및 `Selenium`을 활용한 동적 웹 크롤링으로 수집하였습니다.  
불필요한 목차, 하단 footer 등등 데이터 제거 전처리를 진행하였습니다.

### - Database

법령의 계층 구조를 보존하기 위해 각 법령을 **조(Article)** 및 **항(Clause)** 단위로 나누어 벡터화하여 저장합니다.
